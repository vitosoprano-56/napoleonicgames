Napoleonic Bot Demo

Overview
Napoleonic is a Tap-to-Earn war rebuild game bot originally built as a demo on Replit.  
Players tap to deploy miniature builders who repair monuments in a smoky battlefield.  
The Eiffel Tower begins broken, smoke thickens, and enemy flags appear during timed challenges.  
Progress unlocks new monuments like the Arc de Triomphe, forts, and bridges, while leaderboards rank players as Generals, Colonels, and Captains.

This repository serves as a blueprint for developers and enthusiasts who want to recreate or extend the Napoleonic game bot.  
It includes the core gameplay loop, demo instructions, and integration notes for social sharing and referral systems.

---

Features
- ⚔️ Tap Loop: Each tap adds progress to rebuilding monuments.  
- 🏰 Leaderboards: Military-style ranks track top players and clans.  
- ⏱ Timed Challenges: Smoke thickens, enemies approach, faster tapping required.  
- 👥 Clan Play: Players connect armies and compete collectively.  
- 🎁 Rewards: Demo includes $50 airdrop participation and $2 referral bonuses (via Blink integration).  
- 🌐 Social Media Reach: Designed for sharing across Facebook, Instagram, X, TikTok, Snapchat, LinkedIn, YouTube, Reddit, Pinterest, and Telegram.  

---

Setup
1. Clone the repo  
   `bash
   git clone https://github.com/VitoSoprano/napoleonic-bot-demo.git
   cd napoleonic-bot-demo
   `

2. Install dependencies (Node.js example)  
   `bash
   npm install
   `

3. Run the server  
   `bash
   npm start
   `

4. Connect to Telegram Bot API  
   - Create a bot via BotFather.  
   - Copy your bot token into .env.  
   - Launch the bot and test the tap loop.
